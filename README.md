# The-Starter-Kit-Agent-System-Antigravity-LG-
A modular Flutter skeleton for LG apps. Features a pre-built SSH/KML service and an AI "Brain" in the .agent/ folder. Using Gemini &amp; Antigravity, it teaches AI to automate 3D orbits and multi-screen rig control. Built for GSoC 2026 to turn raw data into immersive experiences. MIT Licensed.
